
#Bangladeshi Payment Gateways for WooCommerce.

== Description ==

Bangladeshi Payment Gateways for WooCommerce. It has some advance feature that will help you to manage payment easily.

### Features

* Pay with QR Code
* Fee for each gateway
* And many more...

### Supported Gateways
* bKash
* Rocket
* Nagad
